微信小程序开发之转发功能

发表于 2017-05-16 | 分类于 微信小程序 | | 阅读

之前无聊的时间调试了一个微信小程序。
利用一个美女图片的api做的一个微信美女图片浏览的小程序，之前也没有做过,后面上线了发现没有转发分享功能，这点感觉很不爽啊，怎么能没有转发这种高逼格的功能呢？

赶紧打开微信开发工具

首先参考了一下:官方文档

然后添加转发的显示按钮

wx.showShareMenu({ withShareTicket: true })
